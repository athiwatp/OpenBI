OpenBI
======

Web-based Business Intelligence system for the rest of us on this planet.

Installation
------------
>$ git clone https://github.com/kookiatsuetrong/openbi.git<br/>
>$ cd openbi<br/>
>$ npm install<br/>
>$ node app
